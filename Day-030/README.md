# Day 30 - DBSCAN Clustering (Density-Based Spatial Clustering)

Today I implemented and evaluated **DBSCAN** (Density-Based Spatial Clustering of Applications with Noise) clustering on synthetic moon-shaped data, and visualized the results with cluster assignments.

---

## Key Learnings

- Learnt about **DBSCAN**, a density-based clustering algorithm that can identify clusters of arbitrary shapes
- Visualized the raw data to understand the spatial distribution and challenges with distance-based clustering
- Built and evaluated **DBSCAN** model with parameters `eps=0.3` and `min_samples=5`
- Understood key DBSCAN parameters: **eps** (neighborhood radius) and **min_samples** (minimum points to form a cluster)
- Visualized the clustering results by plotting data points colored by their assigned cluster labels
