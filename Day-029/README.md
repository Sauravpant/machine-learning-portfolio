# Day 29 - Hierarchical Clustering (Dendrograms & Agglomerative)

Today I implemented and evaluated **Hierarchical (Agglomerative)** clustering on the Iris dataset, visualized relationships using dendrograms, and projected data with PCA for clear cluster visualization.

---

## Key Learnings

- Learnt how to build and evaluate **Hierarchical / Agglomerative** clustering
- Used **PCA** to reduce features to 2 components for visualization of cluster structure
- Constructed **dendrograms** (Ward linkage) to inspect hierarchical relationships and identify plausible cut points
- Performed **Agglomerative Clustering** with `linkage='ward'` and `metric='euclidean'` and examined the resulting cluster labels
