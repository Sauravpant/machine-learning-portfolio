# Machine Learning Portfolio

This repository is a collection of my Machine Learning learning journey, experiments, and projects. It serves as a space where I practice core concepts, explore algorithms, and build small to medium-scale projects.

## Purpose

- Learn Machine Learning from fundamentals to advanced topics  
- Implement concepts through hands-on coding  
- Build practical projects for real-world understanding  
- Document progress and key learnings  
---

## Daily Workflow

Each day is organized with a consistent workflow:

1. **Notebook (`notebooks/`)** – Experiment, implement, and test code
2. **Notes (`README.md`)** – Record key concepts, insights, and observations
3. **Data (`data/`)** – datasets used for experiments

<h2> Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com//machine-learning-portfolio.git
```

<p>2. Create Virtual Environment  (Optional)</p>

```
python -m venv .venv
```

<p>3. Activate virtual environment:</p>

```
# Windows .venv\Scripts\activate
OR
# Linux / Mac source .venv/bin/activate
```

<p>4. Install dependencies:</p>

```
pip install -r requirements.txt
```

<p>5. Launch Jupyter Lab from the root:</p>

```
jupyter lab
```

> All notebooks will use the root .venv kernel for consistent package management.
